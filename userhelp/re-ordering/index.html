Re-ordering

You can use the re-ordering option to quickly repeat the order process to one of your previous orders.

Watch a Flash demonstration on this topic

Re-order

Locate the original order that you want to re-order (Learn how to do that here) and click on the Re-order link.

This will start the same ordering process as before except this time the text and the images in the input form are already filled in. All you need to do is click “Update preview” on each page and fill in the order form.
