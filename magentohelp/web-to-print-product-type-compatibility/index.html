Product type compatibility

Web-to-print administration, Associate, Compatibility, Product types
11/05/2011

There are a few different product types in Magento. Here’s an overview on which ones can be associated with web-to-print templates.

Magento supports the following 6 types of products:

  1. Simple
  2. Grouped
  3. Configurable
  4. Virtual
  5. Bundle
  6. Downloadable

Simple

This is the very basic type of product. Our free web-to-print extension supports this product type and is equipped with a auto-creating feature that allows you to automatically create simple Magento products for all your imported web-to-print templates.

Learn how to populate your store with a combination of simple and virtual web-to-print products.

Grouped

Grouped products allow you to create a new product using one or more existing products in your store. Unfortunately, simple products that have web-to-print templates associated with them cannot be grouped this way nor can a web-to-print template be associated with a grouped product directly.

Configurable

Configurable products are supported. A web-to-print template can only be associated with a configurable product on a super product level. As an important requirement, you must assign the web-to-print template after creating and associating the simple products with the configurable super product. Ex:

  1. Create the configurable product
  2. Quick create the simple products (options) from within the configurable product
  3. Associate the simple products with the configurable one and Save the product
  4. Navigate to Web-to-print templates tab on product settings page and select the template you want associated with that configurable product

Note. Use configurable products if you want to keep track of your inventory. Otherwise, the same can be achieved by adding custom options to a simple product, which is far less complicated to create.

Virtual

Same as for simple products, our free web-to-print extension supports this product type and is equipped with a auto-creating feature that allows you to automatically create virtual Magento products for all your imported web-to-print templates.

Learn how to populate your store with a combination of simple and virtual web-to-print products.

Bundle

Bundle products are not supported and web-to-print templates cannot be associated with them.

Downloadable

Web-to-print templates can be associated with a downloadable product. This, however, may be redundant depending on what you’re trying to achieve. If you want to allow customers to download the personalized print-ready files generated from the web-to-print template, you don’t have to use a downloadable product. This can be done with simple or a virtual products. There are two ways to allow file downloading from customer order details page:

  • Allow output file downloading for specific products only
  • Allow output file downloading for all products in your store
